## 機能一覧

### ランダムで選ぶ
- 抽選対象から `Math.floor(Math.random() * n)` で一人を選択
- 選ばれた感を出すためにランダムで1回選ばれたら、ではなく最初に規定回数に到達した対象を選ぶ

厳選感 | ロジック
---|---
1%  |10,000回選択された対象を抽出する。
10% | 100,000回選択された対象を抽出する。
100% | 1,000,000回選択された対象を抽出する。

- 横線はクリックで好きな場所に線を引くことが可能
